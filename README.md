# Job Trend Analyzer — Streamlit Job Market Dashboard

A Python Streamlit dashboard that collects job listing data from a public job source, stores it in CSV format and visualizes job market trends through filters, charts and summary insights.

## Project Overview

Job Trend Analyzer is a data analytics dashboard designed to explore job market trends in a simple and interactive way. The project fetches job listing data, saves it into a CSV file and displays useful insights such as top job titles, companies, locations and posting trends.

This project was created as a beginner-friendly portfolio project to demonstrate Python data handling, CSV processing, data analysis and Streamlit dashboard development.

## Features

- Job listing data collection from a public job source
- CSV data storage
- Interactive Streamlit dashboard
- Search jobs by keyword
- Filter jobs by company
- Filter jobs by location
- Dashboard summary cards
- CSV data preview
- Download filtered CSV results
- Top job titles visualization
- Top companies visualization
- Top hiring locations visualization
- Posting trend over time

## Tech Stack

- Python
- Streamlit
- pandas
- matplotlib
- requests
- CSV
